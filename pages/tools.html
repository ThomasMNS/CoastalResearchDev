Tools

This page contains details of some of the tools we have developed, that are freely available for your own use.

Grim

GRoyne Impacts on Morphology

Grim is a freely available Python plugin for the open source GIS program QGIS, designed for analysing the impacts of coastal groynes on beach morphology. The tool is easy to use, featuring simple step by step instructions, and can work with a variety of data types.

The main feature of Grim is groyne cell elevation analysis. This requires one or more raster digital elevation models (DEMs) covering a beach area, and the locations of groyne cells (the area between two groynes) covering the same area. These gronyne cells can either be input directly, or alternatively the locations of the groynes themselves can be used, and Grim will delineate the locations of groyne cells for you. A number of vector datatypes are supported for inputting groyne cell locations, including single and multiple polygons, line and point shapefiles.

From this, Grim will calculate the area of each groyne cell, as well as the volume and average elevation of material within it, providing information on the spatial variability of morphology along the groyne field. This can be used, for example, to examine whether groynes are blocking longshore transport, leading to a starvation of sediment further up the coast. Results are output as a CSV file, which can be opened by programs such as Microsoft Excel or LibreOffice Calc for further analysis. Shapefiles containing the results as feature attributes will also be created, which are useful for data visualisation purposes.

In addition to this, Grim can be used to quickly create cross sections through digital elevation models, useful for taking beach profiles from elevation data. The tool takes one or more DEMs, and one or more vector lines, and samples the DEM on this line at user specified intervals. Again, the results are output as a CSV file, and a shapefile which can be used for visualisation.

Get Grim

Click here to download Grim. It can be installed by copying the unzipped folder to your QGIS plugins directory (on Windows, this will be similar to "C:\Users\Thomas.qgis2\python\plugins")

Help

Grim includes a comprehensive help system. For help during use, click the "?" button at the top right of the screen and then click on a UI element for details about what it does. You can also access the help files by clicking the "Help" button, or download them seperately here.

SfM_georef

SfM_georef is a tool developed by Dr Michael James for scaling and geo-referencing structure-from-motion (SfM) point clouds to real-world coordinates, using observations made directly in the SfM image set. It also includes functions for processing the output of ground control point and point precision analyses, which are carried out using Monte Carlo approaches with PhotoScan. For more information, view this page.
